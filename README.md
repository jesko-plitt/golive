# golive

golive is a webserver support package for you go (golang) web application.

It is based on gofiber and gorm.

It provides helpers for:

- minio/s3
- imageproxy
- redis
- csrf protection
- environment variables (configuration)
- logging (zerolog)
- database (gorm)
- webserver / router (gofiber)
